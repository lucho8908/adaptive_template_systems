multidim.SimplicialComplex.make_pers1_rca1¶

SimplicialComplex.make_pers1_rca1(cutoff=-1.0)[source]¶

Run RCA1 and make a 1-dimensional homology.PersDiag for the edge-pairings for cycle generators.

This reruns self.make_pers0(cutoff) again.

Parameters:

cutoff: float

Maximum edge value to use for RCA1 algorithm. Longer edges ignored. (Default: -1, meaning use no edges.)
Returns:

none. Produces self.pers1

Table of edge pairs, similar to a persistence diagram.
