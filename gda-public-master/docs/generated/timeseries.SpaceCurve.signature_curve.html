timeseries.SpaceCurve.signature_curve¶

SpaceCurve.signature_curve()[source]¶

Olver/Boutin signature curve. (kappa, kappa_s, tau, tau_s) (currently planar only)

>>> ts = np.arange(0,12,0.1)
>>> # Line
>>> C = SpaceCurve(tn = np.arange(ts.shape[0]),
...                px = 5*ts + 3, py=2*ts + 5)
>>> sig = C.signature_curve()
>>> np.allclose(sig, 0.0, atol=1.5e-7)
True
>>> ts = np.arange(0,12,0.1)
>>> # Circle with constant speed
>>> C = SpaceCurve(tn = np.arange(ts.shape[0]),
...                px = np.cos(ts),
...                py = np.sin(ts))
>>> sig = C.signature_curve()
>>> sig.shape
(116, 2)
>>> np.allclose(sig[:,0], 1.)
True
>>> np.allclose(sig[:,1], 0.)
True

>>> ts = np.arange(0,12,0.1)
>>> # Circle with varying speed
>>> C = SpaceCurve(tn = np.arange(ts.shape[0]),
...                px = np.cos(ts**2),
...                py = np.sin(ts**2))
>>> sig = C.signature_curve()
>>> sig.shape
(116, 2)
>>> np.allclose(sig[:,0], 1.)
True
>>> np.allclose(sig[:,1], 0.)
True

>>> ts = np.arange(1,13,0.01)
>>> # A Spiral
>>> C = SpaceCurve(tn = np.arange(ts.shape[0]),
...                px = np.exp(0.75*ts)*np.cos(ts),
...                py = np.exp(0.75*ts)*np.sin(ts))
>>> sig = C.signature_curve()
>>> sig.shape
(1196, 2)
>>> np.allclose(sig[:,0],  np.exp(-0.75*ts[2:-2]), atol=0.1)
True
>>> np.allclose(sig[:,1]*np.exp(1.5*ts[2:-2]), -12./25., atol=0.01)
True
